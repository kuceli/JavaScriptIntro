1. What runs JavaScript outside the browser: Node
2. 5 things Javascript can do: Add interactive behaviour to web pages, develop games, create web and mobile applications, develop server applications, and create real-time networking applications.
3. When ECMAScript was first released: 1997
4. How to log to the console: To log to console, Open up Chrome, Right Click on empty area and go to inspect. Then, select the console tab from the chrome developer tools. The console is then opened up and any valid Javascript can be written. For instance, Typing console.log(‘Hello World’); would output the Hello World message on the console.
5. Tech fields that use JavaScript: Front-end development, web application development, back-end development, mobile app development, systems admin
6. Difference between statically typed and dynamically typed languages: Statically typed languages perform type checking at compile time e.g Java, C, and C++. While, dynamically-typed languages perform type checking at runtime e.g Python, PHP, and JavaScript.
7. Difference between the JavaScript data types ‘null’ and ‘undefined': 'Null' indicates the absence of a value for a variable. 'Undefined', on the other hand, indicates the absence of the variable itself.
